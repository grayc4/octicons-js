Octicons-JS

A lightweight JavaScript library for using GitHub Octicons

View on GitHub

About

Octicons-JS provides easy access to GitHub's Octicons for use in JavaScript projects. It includes 300+ SVG icons with a simple, framework-agnostic API.

This library offers:

  • 300+ SVG Icons - The complete GitHub Octicons set
  • Zero Dependencies - No external libraries required
  • Framework Agnostic - Works with any JavaScript project
  • TypeScript Support - Fully typed API with autocompletion
  • Tree-Shaking Support NEW - Import only what you need
  • Simple API - Straightforward functions for creating icons and buttons
  • Customizable - Control size, color, and styling with simple props

Installation

Step #1: Package installation 
npm install octicons-js
Step #2: Choose preferred import method...
    Method #1
    // Import the previously installed package 'octicon-js' in a .js page to access all icons
import { icons, octicon, octiconBtn, applyStyles } from 'octicons-js';

// Required default styles (important)
applyStyles();
    Method #2
    // Import only the icons you need (tree shaking)
import { alert, download, octicon, applyStyles } from 'octicons-js';

// Required default styles (important)
applyStyles();
    Method #3
    // Import from CDN url. The one below is the most up to date CDN url.
import { icons, octicon, octiconBtn, applyStyles } from 'https://cdn.jsdelivr.net/npm/octicons-js@1.4.2/+esm';

// Required default styles (important)
applyStyles();

Icons (default style)

import { alertFill, search, download, starFill, heartFill, applyStyles } from 'octicons-js';

// Insert basic icons with default styling

[
    { icon: icons.alertFill, size: 18 },
    { icon: icons.search, size: 18 },
    { icon: icons.download, size: 18 },
    { icon: icons.starFill, size: 18 },
    { icon: icons.heartFill, size: 18 }
].forEach(({ icon, size }) => {
    basicIcons.appendChild(octicon(icon, size));
});

// Option 2: Using individually imported icons (enables tree-shaking)
basicIcons.appendChild(octicon(alertFill, 18));
basicIcons.appendChild(octicon(search, 18));
basicIcons.appendChild(octicon(download, 18));
basicIcons.appendChild(octicon(starFill, 18));
basicIcons.appendChild(octicon(heartFill, 18));

Interactive Button Examples
Icon buttons with click animation
import { copy, sync, octiconBtn } from 'octicons-js';

// Interactive buttons with loading animation
const buttonSizes = document.getElementById('buttonSizes');

// Create buttons with different sizes
[
    { size: 12, buttonSize: 28 },
    { size: 14, buttonSize: 32 },
    { size: 16, buttonSize: 36 },
    { size: 20, buttonSize: 40 }
].forEach(({ size, buttonSize }) => {
    const btn = octiconBtn(copy, size, buttonSize);

    btn.addEventListener('click', () => {
        const originalIcon = btn.querySelector('svg');
        originalIcon.style.display = 'none';

        // Create a temporary spinner icon
        const spinnerIcon = octiconBtn(sync, size).querySelector('svg');
        spinnerIcon.classList.add('spin');
        btn.prepend(spinnerIcon);

        // After 1 second, revert to original icon
        setTimeout(() => {
            spinnerIcon.remove();
            originalIcon.style.display = '';
        }, 1000);
    });

    buttonSizes.appendChild(btn);
});

Interactive Label Buttons
Label buttons with loading animations
import { rocket, flame, heartFill, sync, octiconLabelBtn } from 'octicons-js';

// Interactive label buttons with loading animation
const customLabelButtons = document.getElementById('customLabelButtons');

// Create interactive buttons
[
    { icon: rocket, label: 'Launch', fill: '#0969da' },
    { icon: flame, label: 'Trending', fill: '#ff7b00' },
    { icon: heartFill, label: 'Favorite', fill: '#e3242b' }
].forEach(({ icon, label, fill }) => {
    const btn = octiconLabelBtn(icon, label, 'dark', 16, true, fill);
    btn.style.transition = 'all 0.2s ease';

    // Add loading animation on click
    btn.addEventListener('click', () => {
        const originalIcon = btn.querySelector('.octicon');
        const originalText = btn.querySelector('.icon-label').textContent;

        // Replace icon and text
        originalIcon.remove();
        btn.querySelector('.icon-label').textContent = 'Loading...';

        // Create spinner icon
        const spinnerIcon = octiconLabelBtn(sync, '', 'dark', 16, true, fill).querySelector('.octicon');
        spinnerIcon.classList.add('spin');
        btn.prepend(spinnerIcon);

        // Disable button during loading
        btn.disabled = true;
        btn.style.opacity = '0.8';

        // After 2 seconds, revert to original
        setTimeout(() => {
            spinnerIcon.remove();
            btn.prepend(octiconLabelBtn(icon, '', 'dark', 16, true, fill).querySelector('.octicon'));
            btn.querySelector('.icon-label').textContent = originalText;
            btn.disabled = false;
            btn.style.opacity = '1';
        }, 2000);
    });

    customLabelButtons.appendChild(btn);
});

Icon Groups
Interactive rating system and status indicators
import { starFill, star, checkCircleFill, octicon } from 'octicons-js';

// Create interactive star rating system
const iconGroups = document.getElementById('iconGroups');
const ratingContainer = document.createElement('div');
ratingContainer.className = 'rating-container';
let currentRating = 3;

function renderStars(rating) {
    ratingContainer.innerHTML = '';
    for (let i = 1; i <= 5; i++) {
        const iconElement = octicon(
            i <= rating ? starFill : star,
            24,
            'dark',
            i <= rating ? '#ffab00' : '#d0d7de'
        );

        iconElement.classList.add('star-rating-icon');
        iconElement.addEventListener('click', () => {
            currentRating = i;
            renderStars(currentRating);
        });

        iconElement.addEventListener('mouseover', () => {
            renderStars(i);
        });

        ratingContainer.appendChild(iconElement);
    }
}

ratingContainer.addEventListener('mouseout', () => {
    renderStars(currentRating);
});

renderStars(currentRating);
iconGroups.appendChild(ratingContainer);

// Status indicator
const statusContainer = document.createElement('div');
statusContainer.style.display = 'flex';
statusContainer.style.alignItems = 'center';
statusContainer.style.gap = '8px';
statusContainer.style.marginLeft = '32px';
statusContainer.style.backgroundColor = '#f6f8fa';
statusContainer.style.padding = '8px 12px';
statusContainer.style.borderRadius = '6px';
statusContainer.style.border = '1px solid #d0d7de';

const checkIcon = octicon(checkCircleFill, 16, 'dark', '#1a7f37');

const statusLabel = document.createElement('span');
statusLabel.textContent = 'All checks passed';
statusLabel.style.fontWeight = '500';
statusLabel.style.color = '#1a7f37';

statusContainer.appendChild(checkIcon);
statusContainer.appendChild(statusLabel);
iconGroups.appendChild(statusContainer);

Tree-Shaking Benefits NEW

Learn how tree-shaking helps reduce bundle size

With the new tree-shaking support, you can significantly reduce your bundle size by only importing the icons you need:

Before: Without Tree-Shaking
import { icons, octicon } from 'octicons-js';

// This imports the entire icon collection (300+ icons)
const alertIcon = octicon(icons.alert);
After: With Tree-Shaking
import { alert, octicon } from 'octicons-js';

// This only imports the alert icon
const alertIcon = octicon(alert);

Bundle size comparison:

  • Without tree-shaking: ~200KB (all icons)
  • With tree-shaking: ~1-2KB per icon

Full API

applyStyles()

Applies the required CSS styles into the document's head. Call this once at the beginning of your app.

octicon(icon, size?, style?, fill?)

Creates an SVG icon element.

icon: IconFunction | Element - The icon function or element

size: Number - Size in pixels (default: 16)

style: String - "dark" or "light" variant (default: "dark")

fill: String - Fill color (default: null, uses currentColor)

Returns: HTMLElement

octiconBtn(icon, size?, buttonSize?, style?, fill?)

Creates a square button containing an icon.

icon: IconFunction | Element - The icon function or element

size: Number - Size of the icon (default: 16)

buttonSize: Number - Size of the button (default: null - auto)

style: String - "dark" or "light" variant (default: "dark")

fill: String - Fill color (default: null, uses currentColor)

Returns: HTMLButtonElement

octiconLabelBtn(icon, label, style?, size?, iconFirst?, fill?)

Creates a button with both an icon and a text label.

icon: IconFunction | Element - The icon function or element

label: String - Text label for the button

style: String - "dark" or "light" variant (default: "dark")

size: Number - Size of the icon (default: 16)

iconFirst: Boolean - Whether to show icon before text (default: true)

fill: String - Fill color (default: null, uses currentColor)

Returns: HTMLButtonElement

icons

Object containing all available icons as functions.

import { icons } from 'octicons-js';

// Access icons by name
const alertIcon = icons.alert;
const starIcon = icons.star;

Individual Icon Imports NEW

Import icons individually to enable tree-shaking.

import { alert, star, download } from 'octicons-js';

// Use imported icons directly
const alertIcon = octicon(alert);
const starIcon = octicon(star);
const downloadIcon = octicon(download);
