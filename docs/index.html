octicons-js

GitHub Octicons for JavaScript projects with a simple, intuitive API. Create icon elements and buttons with minimal code. Zero dependencies, fully TypeScript-compatible, and ready for both Node.js and browser environments. Includes all official GitHub icons with light/dark styling options.

Installation

npm install octicons-js

Usage

Simplified API (Recommended)

Use these helper functions for the easiest and most intuitive way to work with Octicons:

icon(name, options) New

Create a standalone icon using just the icon name:

import { icon } from 'octicons-js';

// Create a 16px alert icon with dark style
const alertIcon = icon('alert');
document.body.appendChild(alertIcon);

// Create a 24px gear icon with light style
const gearIcon = icon('gear', { size: 24, style: 'light' });
document.body.appendChild(gearIcon);
iconButton(name, options) New

Create a square button containing just an icon:

import { iconButton } from 'octicons-js';

// Create a 16px x icon button (32px square button)
const closeBtn = iconButton('x', { buttonSize: 32 });
document.body.appendChild(closeBtn);

// Create a large search button with custom styling
const searchBtn = iconButton('search', {
  size: 24,             // Icon size
  buttonSize: 48,       // Button dimensions
  style: 'light'        // Icon color
});
document.body.appendChild(searchBtn);
labelButton(name, label, options) New

Create a button with both an icon and text label:

import { labelButton } from 'octicons-js';

// Create a settings button with default options
const settingsBtn = labelButton('gear', 'Settings');
document.body.appendChild(settingsBtn);

// Create a download button with icon after text
const downloadBtn = labelButton('download', 'Download ZIP', {
  size: 16,             // Icon size
  style: 'dark',        // Icon color
  iconFirst: false      // Place icon after text
});
document.body.appendChild(downloadBtn);
buttons New

Pre-configured buttons for common icons:

import { buttons } from 'octicons-js';

// Add buttons with a single function call
document.body.appendChild(buttons.gear('Settings'));
document.body.appendChild(buttons.plus('Add Item'));
document.body.appendChild(buttons.file('View File'));

// Icon-only buttons (no label)
document.body.appendChild(buttons.gearIcon());
document.body.appendChild(buttons.alertIcon());

Core API

Access raw SVG strings and create elements with more control:

Octicons

Object containing all icon SVG strings indexed by name and size:

import Octicons from 'octicons-js';

// Each icon is available in multiple sizes (12, 16, 24, 48, 96)
const alertSvg = Octicons.octiconAlert16;
const gearSvg = Octicons.octiconGear24;

// Always check if the icon exists
if (alertSvg) {
  console.log('Alert icon SVG:', alertSvg);
}
octicon(svg, size, style)

Creates a span element with the icon SVG inside:

import Octicons, { octicon } from 'octicons-js';

const fileSvg = Octicons.octiconFile16;

if (fileSvg) {
  // octicon(svgString, size, style)
  const fileIcon = octicon(fileSvg, 16, 'dark');
  document.body.appendChild(fileIcon);
}
octiconBtn(svg, style, size, buttonSize)

Creates a square button with a centered icon:

import Octicons, { octiconBtn } from 'octicons-js';

const zapSvg = Octicons.octiconZap16;

if (zapSvg) {
  // octiconBtn(svgString, style, iconSize, buttonSize)
  const zapButton = octiconBtn(zapSvg, 'dark', 16, 32);
  document.body.appendChild(zapButton);
}
octiconLabelBtn(svg, label, style, size, iconFirst)

Creates a button with an icon and text label:

import Octicons, { octiconLabelBtn } from 'octicons-js';

const folderSvg = Octicons.octiconFileDirectory16;

if (folderSvg) {
  // octiconLabelBtn(svgString, label, style, iconSize, iconFirst)
  const folderButton = octiconLabelBtn(folderSvg, 'My Folder', 'dark', 16, true);
  document.body.appendChild(folderButton);
}

TypeScript Support

The package includes TypeScript definitions for all functions and interfaces:

import { buttons, icon, iconButton, labelButton, IconOptions } from 'octicons-js';

// Options interface for customizing icons
const options: IconOptions = {
  size: 24,
  style: 'dark'
};

// Create and use components with type safety
const alertIcon = icon('alert', options);
document.body.appendChild(alertIcon);

// Pre-configured buttons with TypeScript support
const settingsBtn = buttons.gear('Settings');
document.body.appendChild(settingsBtn);
