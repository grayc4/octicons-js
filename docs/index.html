Octicons-JS

A lightweight JavaScript library for using GitHub Octicons

View on GitHub

About

Octicons-JS provides easy access to GitHub's Octicons for use in JavaScript projects. It includes 300+ SVG icons with a simple, framework-agnostic API.

This library offers:

  • 300+ SVG Icons - The complete GitHub Octicons set
  • Zero Dependencies - No external libraries required
  • Framework Agnostic - Works with any JavaScript project
  • TypeScript Support - Fully typed API with autocompletion
  • Simple API - Straightforward functions for creating icons and buttons

Installation

npm install octicons-js

Or using yarn:

yarn add octicons-js

Quick Start

import { icon, iconButton, applyStyles } from 'octicons-js';

applyStyles();

const downloadIcon = icon('download', { size: 24, fill: 'blue' });
document.body.appendChild(downloadIcon);

const searchButton = iconButton('search', {
  label: 'Search',
  size: 16,
  fill: 'currentColor'
});
document.body.appendChild(searchButton);

Full API

applyStyles()

Injects the required CSS styles into the document's head. Call this once at the beginning of your app.

icon(name, options?)

Creates an SVG icon element.

name: String - The name of the icon

options: Object - Optional parameters

  • size: Number - Size in pixels (default: 16)
  • fill: String - Fill color (default: 'currentColor')
  • style: String - Additional inline styles

iconButton(name, options?)

Creates a button containing an icon.

name: String - The name of the icon

options: Object - Optional parameters

  • size: Number - Size of the icon (default: 16)
  • fill: String - Color of the icon (default: 'currentColor')
  • label: String - Text label (makes the button non-square)
  • bgSize: Number - Size of the button when square (default: null - auto)
  • bgColor: String - Background color (default: system button color)
